Q1: Write a YACC program to evaluate an expression consisting of the following operators.
+, -, *, /, Unary Minus (-), sin, cos, tan, log, exponent etc.

Example:
1. 2 + 5 * 4
2. sin(60) ^ 2
3. 2 + sin(90) * log(102-2) * (5-2)
4. sin(cos(0) + 59)


Output:
1. 22
2. .75
3. 8
4. sqrt(3)/2
